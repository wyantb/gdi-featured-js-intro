Building Block: Variables

Just like 'x' in algebra, a variable is a named container for a value

To declare (create) a variable, just type the word const and the variable name.

const numberOfKittens = 5;

It is a good idea to give your variable a starting value. This is called initializing the variable.

Building Block: Variable Data Types

  • string string of characters 
    const userName = 'Jane Lane';
  • number integer or floating point 
    const myAge = 30;
  • boolean true or false 
    const catsAreBest = true;
  • null an explicitly empty value 
    const goodPickupLines = null;
  • array a list of values - more advanced! 
    const someNumbers = [1, 2, 3];
console.log(someNumbers[0], someNumbers[1], someNumbers[2]);
console.log(someNumbers.join(", and also: "));
  • object a container of named values - yet more advanced! 
    const personJane = { firstName: 'Jane', lastName: 'Lane' };
console.log(personJane.firstName);

Building Block: Methods

Methods *do* things

You've seen one method already: console.log, which prints to the console

const numberOfKittens = 5;
console.log(numberOfKittens);

Building Block: Methods 2

Here's a new method: Math.random(), that returns a decimal between 0 and 1

Try running this one multiple times!

const randomNumber = Math.random();
console.log(randomNumber);

Building Block: Methods 3

And you can make your own functions

function greetPerson(name) {
    const greeting = "Hello, " + name;
    console.log(greeting);
}
console.log(greetPerson("Jane Doe"));

Building Block: Control Statements - If

Try running this one multiple times!

const randomNumber = Math.random() * 5;
const isUnderThree = randomNumber < 3;
console.log(isUnderThree);
if (isUnderThree) {
    console.log("The number is under 3");
} else {
    console.log("The number is above 3");
}

Building Block: Arithmetic Operators

Once you have numbers, you can do math with them!

Example Name Result
-a Negation Opposite of a.
a + b Addition Sum of a and b.
a - b Subtraction Difference of a and b.
a * b Multiplication Product of a and b.
a / b Division Quotient of a and b.
a % b Modulus Remainder of a divided by b.

Let's Develop It

Create a program to calculate the tip at a restaurant. It should:

  • Have variables for the bill pre-tip and the tip percentage.
  • Calculate the total bill.
  • Output a sentence like "Your total bill, with tip, is $14.75".
  • Bonus: Use toFixed() to round the bill total to 2 decimals.
